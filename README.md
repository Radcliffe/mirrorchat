# MirrorChat

MirrorChat is a local-only web app that mirrors the ChatGPT layout while letting a single user write both sides of a conversation.

## Why it exists

- Helps people think through problems by structured self-talk.
- Acts as a lightweight protest against over-reliance on AI chat tools.
- Runs entirely in the browser with no backend.

## Features

- Default chat title is **"Untitled Chat"** and can be renamed.
- Left panel includes **New chat**, **Search chats**, and **Your chats** history.
- Previous chats are persisted in `localStorage`.
- Message composer lets you add entries as either **You** or **Mirror**.

## Run

Open `index.html` in any modern browser.
